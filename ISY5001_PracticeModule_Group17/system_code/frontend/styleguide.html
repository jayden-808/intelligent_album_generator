How do we arrange the album?

Read information from Exif

Exchangeable image file format (English: Exchangeable image file format, officially referred to as Exif) is a file format specially set for photos taken by digital cameras. It can record the attribute information and shooting data of digital photos.

Get time and aspect ratio

Obtain the aspect ratio and shooting time information by uploading the Exif information of the photo.

Get the color components of the image

Convert image to RGB format. Convert the image data to a numpy array and reshape it into N x 3 form. Here N is the number of pixels in the image, and 3 represents the three channels of RGB. Use KMeans clustering to determine the dominant color in these pixels.

Background recognition

The place365 model is called to identify the background of the photo and use it for album layout.

Genetic algorithm for album layout

Use genetic algorithms to perform intelligent layout and cropping based on the background, color, shooting time, and size of the photo.

IOS and Windows systems

Suitable for HEIC format photos taken by Apple mobile phones. Finally export the PDF album file.
